# Minhoteca

## Descrição do Projeto

A Minhoteca é uma plataforma de biblioteca virtual open-source, desenvolvida para facilitar o empréstimo e a gestão de livros de uma biblioteca particular. Nosso objetivo é democratizar o acesso à leitura, permitindo que comunidades e indivíduos compartilhem seus acervos de forma organizada e eficiente.

## Funcionalidades Principais

- **Catálogo de Livros:** Gerenciamento completo de títulos, autores, gêneros e status de disponibilidade.
- **Empréstimos e Devoluções:** Sistema intuitivo para registrar empréstimos, devoluções e controlar prazos.
- **Perfis de Usuários:** Cadastro e gestão de membros da biblioteca, com histórico de empréstimos.
- **Busca Avançada:** Ferramentas de busca para encontrar livros rapidamente.
- **Interface Amigável:** Design responsivo e fácil de usar para todos os tipos de dispositivos.

## Tecnologias Utilizadas

A Minhoteca é desenvolvida utilizando tecnologias modernas e populares no ecossistema JavaScript, focando em performance e qualidade de código.

- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Testes:** [Jest](https://jestjs.io/) para testes unitários e de integração.
- **Qualidade de Código:** Análise estática de código com [SonarQube](https://www.sonarqube.org/).
